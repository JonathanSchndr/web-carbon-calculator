## How to Create Issues from These Templates

### Step 1: Navigate to Issues
Go to: https://github.com/JonathanSchndr/web-carbon-calculator/issues

### Step 2: Click "New Issue"
You'll see a list of available templates

### Step 3: Select a Template
Click "Get started" next to the feature you want to create an issue for

### Step 4: Review & Submit
- The issue will be pre-filled with all the details
- You can modify or add to the template as needed
- Click "Submit new issue"

That's it! The issue is created with all the comprehensive details included.
